## 2. 订单管理 - 提单

### 视频信息
- **时长**: 4-5 分钟
- **目标**: 掌握提单的创建、编辑和管理

### 脚本内容

**[开场 - 0:00]**
> 本视频介绍提单管理功能，这是物流业务的核心操作。

**[进入提单列表 - 0:10]**
> 点击左侧导航「订单管理」→「提单」进入提单列表页面。
> 
> 页面顶部是搜索和筛选区域，可以按：
> - 提单号搜索
> - 客户名称筛选
> - 日期范围筛选
> - 状态筛选

**[查看提单列表 - 0:40]**
> 列表展示所有提单信息：
> - 提单号（点击可查看详情）
> - 客户名称
> - 起运港/目的港
> - 货物件数/重量
> - 状态标签
> - 创建时间
> - 操作按钮

**[创建新提单 - 1:10]**
> 点击右上角「新建提单」按钮。
> 
> 填写提单基本信息：
> 1. 选择客户（支持搜索）
> 2. 输入提单号
> 3. 选择运输方式（海运/空运/铁路）
> 4. 填写起运港和目的港
> 5. 选择预计到港日期

**[填写货物信息 - 2:00]**
> 切换到「货物信息」标签页：
> 1. 点击「添加货物」
> 2. 填写品名、数量、重量
> 3. 输入 HS Code（可搜索）
> 4. 填写货值和币种
> 5. 可继续添加多个货物

**[上传附件 - 2:45]**
> 切换到「附件」标签页：
> - 点击上传区域或拖拽文件
> - 支持上传提单扫描件、装箱单、发票等
> - 文件自动保存到云端

**[保存提单 - 3:15]**
> 确认信息无误后，点击「保存」按钮。
> 系统会自动生成提单号并跳转到详情页。

**[编辑和状态管理 - 3:40]**
> 在提单详情页可以：
> - 点击「编辑」修改信息
> - 点击状态按钮更新提单状态
> - 查看操作历史记录

**[结尾 - 4:10]**
> 以上就是提单管理的完整流程。如有问题，请点击帮助按钮查看更多信息。

## 3. 单证管理 - 货物导入

### 视频信息
- **时长**: 3-4 分钟
- **目标**: 学会通过 Excel 批量导入货物数据

### 脚本内容

**[开场 - 0:00]**
> 本视频介绍如何通过 Excel 批量导入货物数据，大幅提升数据录入效率。

**[进入货物导入页面 - 0:10]**
> 点击「单证管理」→「货物导入」进入导入页面。

**[下载模板 - 0:25]**
> 首先下载 Excel 模板：
> 1. 点击「下载模板」按钮
> 2. 打开模板文件，查看列说明
> 3. 按照格式填写货物数据

**[模板说明 - 0:45]**
> 模板包含以下字段：
> - 提单号（必填）
> - 品名（必填）
> - 英文品名
> - 数量、单位（必填）
> - 毛重、净重
> - HS Code
> - 货值、币种
> - 原产国

**[上传文件 - 1:15]**
> 填写完成后，上传 Excel 文件：
> 1. 点击上传区域选择文件
> 2. 或直接拖拽文件到上传区域
> 3. 系统自动解析文件内容

**[数据预览 - 1:40]**
> 上传后进入数据预览页面：
> - 检查解析结果是否正确
> - 红色标记表示数据有问题
> - 可以在此页面直接修改

**[确认导入 - 2:10]**
> 确认数据无误后：
> 1. 点击「确认导入」按钮
> 2. 系统开始处理数据
> 3. 显示导入进度条
> 4. 完成后显示导入结果统计

**[查看结果 - 2:40]**
> 导入完成后可以：
> - 查看成功/失败数量
> - 下载失败记录（如有）
> - 点击「查看货物」跳转到货物列表

**[结尾 - 3:10]**
> 货物导入功能可以大幅提高工作效率，建议批量数据都使用此功能。

## 4. 单证管理 - HS匹配审核

### 视频信息
- **时长**: 3-4 分钟
- **目标**: 了解 HS Code 自动匹配和人工审核流程

### 脚本内容

**[开场 - 0:00]**
> 本视频介绍 HS Code 匹配审核功能，确保货物海关编码的准确性。

**[进入审核页面 - 0:10]**
> 点击「单证管理」→「HS匹配审核」进入审核列表。

**[理解匹配状态 - 0:25]**
> 列表显示待审核的货物：
> - 🟢 高置信度：AI 匹配置信度 > 90%
> - 🟡 中置信度：置信度 70-90%
> - 🔴 低置信度：置信度 < 70%，需人工确认

**[查看匹配详情 - 0:50]**
> 点击货物行查看详情：
> - 货物品名和描述
> - AI 推荐的 HS Code
> - 匹配置信度百分比
> - 相似商品参考

**[审核操作 - 1:20]**
> 审核时可以：
> 1. **确认**：AI 匹配正确，点击「确认」
> 2. **修改**：需要更换 HS Code
>    - 点击「修改」按钮
>    - 搜索或手动输入正确的 HS Code
>    - 点击「保存」
> 3. **跳过**：暂时不处理，点击「跳过」

**[批量操作 - 2:00]**
> 对于高置信度的匹配：
> 1. 勾选多个货物
> 2. 点击「批量确认」
> 3. 一次性完成审核

**[查看匹配记录 - 2:30]**
> 系统会记录所有匹配历史：
> - 点击「匹配记录库」查看
> - 相同品名会自动使用历史记录
> - 提高后续匹配准确率

**[结尾 - 3:00]**
> HS 匹配审核是清关的关键步骤，请仔细核对确保准确性。

## 5. 单证管理 - 税费计算

### 视频信息
- **时长**: 3-4 分钟
- **目标**: 了解如何计算进口关税和增值税

### 脚本内容

**[开场 - 0:00]**
> 本视频介绍税费计算功能，帮助您快速计算进口税费。

**[进入税费计算 - 0:10]**
> 点击「单证管理」→「税费计算」进入计算页面。

**[选择提单 - 0:20]**
> 首先选择要计算税费的提单：
> - 在下拉框中搜索提单号
> - 或从列表中选择

**[查看货物清单 - 0:40]**
> 系统自动加载提单下的所有货物：
> - 品名和 HS Code
> - 数量和货值
> - 原产国信息

**[自动计算 - 1:00]**
> 点击「计算税费」按钮：
> - 系统根据 HS Code 查询税率
> - 自动计算关税金额
> - 自动计算增值税金额
> - 显示总税费

**[查看计算明细 - 1:30]**
> 展开计算明细：
> - 关税税率和计算公式
> - 增值税税率和计算公式
> - 反倾销税（如适用）
> - 优惠税率说明

**[调整参数 - 2:00]**
> 如需调整计算参数：
> - 修改货值（CIF 价格）
> - 更换 HS Code
> - 选择适用的贸易协定
> - 重新计算

**[导出报告 - 2:30]**
> 点击「导出」生成税费报告：
> - 支持 PDF 和 Excel 格式
> - 包含完整计算明细
> - 可发送给客户确认

**[结尾 - 3:00]**
> 税费计算功能帮助您快速准确地估算进口成本。
