# 快速开始指南

## 文件说明

这些是已经编译打包好的前端文件，可以直接部署使用。

## 快速部署（3 步）

### 1. 复制文件到服务器

```bash
# 将所有文件复制到 web 服务器目录
sudo cp -r * /var/www/sysafari-logistics/
```

### 2. 配置 Nginx

```bash
# 复制配置文件
sudo cp nginx.conf.example /etc/nginx/sites-available/sysafari-logistics

# 编辑配置文件，修改域名和路径
sudo nano /etc/nginx/sites-available/sysafari-logistics

# 启用站点
sudo ln -s /etc/nginx/sites-available/sysafari-logistics /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

### 3. 访问应用

打开浏览器访问：`http://your-server-ip` 或 `http://your-domain`

## 或使用部署脚本

```bash
# 运行部署脚本
./deploy.sh /var/www/sysafari-logistics
```

## 重要文件

- `index.html` - 应用入口文件
- `umi.js` - 主 JavaScript 文件
- `umi.css` - 主样式文件
- `*.js` - 代码块文件（不要删除）
- `*.chunk.css` - 样式块文件（不要删除）

## 常见问题

**Q: 页面显示空白？**
A: 检查浏览器控制台，确认所有 JS/CSS 文件都能正常加载

**Q: 路由跳转 404？**
A: 确保 Nginx 配置了 `try_files $uri $uri/ /index.html;`
